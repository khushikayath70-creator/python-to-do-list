# ✅ TaskFlow — To-Do List App

A clean, file-based To-Do list app built with **Python** and **Streamlit**. No database — tasks are stored in a simple `.txt` file, but the UI feels like a modern productivity app.

## Features
- ➕ Add tasks instantly
- 👀 View all tasks with a live progress bar and stats (total / pending / completed)
- ✔️ Mark tasks complete / undo completion
- 🗑️ Delete tasks
- 🔍 Filter by All / Pending / Completed
- 💾 Persistent storage via plain text file (`to_dolist.txt`)

## Tech Stack
- Python
- Streamlit
- File I/O (no external database required)

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
